Passo a Passo para Procedimentos Internos de Suporte a API

1. Verificação do Servidor CERMOB

Acesse o link do servidor CERMOB para verificar se está ativo:

  • https://certeduc.com:45143/system

2. Realizar login na API via Postman

Utilize o Postman para realizar o login na API utilizando o CNPJ/CPF conforme documentação fornecida.

3. Procedimentos válidos para RD e GD

Certifique-se de que os procedimentos de teste são válidos para os módulos RD (Receita Digital) e GD (Gestão de Documentos).

4. Acessar o portal e executar uma ação

Acesse o portal correspondente à API e execute alguma ação que envolva interação com os serviços providos pela API.

5. Coleta de informações e Testes Realizados

Após realizar todas as ações acima, reúna todas as informações relevantes e detalhes dos testes realizados.

6. Repassar informações para Nível 3 (N3) - CERMOB

Compile todas as informações coletadas e os resultados dos testes em um ticket ou relatório para serem repassados ao suporte de Nível 3 (N3) da CERMOB.
